# ikkoa

Landing page del estudio de pintura **ikkoa** — construida con Next.js, Tailwind CSS y preparada para integrar Supabase.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- Supabase (próximo)

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Secciones

- **HOME** — hero con marca ikkoa
- **Visual Stories** — obras seleccionadas
- **Expressions** — manifiesto artístico
- **Behind the Canvas** — el proceso en el estudio
- **Say Hello** — contacto
